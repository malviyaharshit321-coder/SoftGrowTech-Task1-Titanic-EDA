# Task 1: Titanic Survival Analysis (EDA)
Part of my Data Science internship at SoftGrowTech.

## 📊 Project Overview
This project involves performing Exploratory Data Analysis (EDA) on the Titanic dataset to discover key factors that influenced passenger survival.

## 🛠️ Tech Stack
- **Python** (Pandas, Seaborn, Matplotlib)
- **Environment:** Google Colab

## 💡 Key Insights
- **Gender:** Female passengers had a significantly higher survival rate.
- **Fare:** Passengers who paid higher fares were more likely to survive.
- **Correlation:** Strong relationships found between Pclass, Sex, and Survival.
